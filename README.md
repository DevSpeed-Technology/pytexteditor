# Python Text Editor

A small desktop text editor built with Python's standard `tkinter` library.

## Run

```bash
python3 text_editor.py
```

## Features

- New, open, save, and save as
- Undo and redo
- Cut, copy, paste
- Case-insensitive find with highlighted matches
- Status bar with file path, cursor position, word count, and character count
- Unsaved-change prompt on new/open/exit
